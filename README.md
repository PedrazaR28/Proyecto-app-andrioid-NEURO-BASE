# NEUROBASE
Aplicación Android nativa
---

**NeuroBase** es una aplicación de **preguntas y respuestas** tipo juego educativo.

## Instalación y ejecución del proyecto 

## Requisitos técnicos

- Android Studio
- Android SDK
- Gradle
- Emulador Android o dispositivo físico
- Git (opcional)

### **Paso 1: Obtener el proyecto**

El proyecto puede obtenerse de **dos formas**:

**Opción A: Clonar el repositorio (recomendado)**  
bash
git clone https://github.com/PedrazaR28/Proyecto-app-andrioid-NEURO-BASE.git

### Opción B: Descargar el proyecto sin usar Git

1. Ingresar al repositorio del proyecto en GitHub desde el navegador.
2. Hacer clic en el botón **Code**.
3. Seleccionar la opción **Download ZIP**.
4. Esperar a que finalice la descarga del archivo.
5. Descomprimir el archivo ZIP en una carpeta local.
6. Abrir **Android Studio**.
7. Seleccionar **File → Open**.
8. Buscar y seleccionar la carpeta descomprimida **NEUROBASE**.
9. Presionar **OK** y esperar a que el proyecto cargue.
